ブログ立ち上げてみました

2017-09-02
#初回, Hexo, ブログ

ブログ作ってみました

以前から知人に

  • 「HPやブログを立ち上げたいが、どうしたらよいか」

という質問を受けることがあって、そのたびに

  • 「かんたんに作れるやつあるよ」
    と言っておきながら参考文献を送るだけだったので、今日は試してで自分で立ち上げてみました。

お金をかけたくなかったので、Gitを利用するのは最初から決めていて、
静的サイトジェネレータ（サーバが不要なもの）でいくつか候補があがりましたが、
今回は Hexo をつかってつくってみました！

参考にしたのが、↓のページ

  • Node.js製の静的サイトジェネレータ「Hexo」で無料ブログ開発 vol.1
  • Node.js製の静的サイトジェネレータ「Hexo」で無料ブログ開発 vol.2

実際にやってみて、やはり簡単でしたが、
まあGitやコマンドラインを使った人がない人には、なかなかハードルが高いな…というのが、心象。

SSHのエラーではまったり、Mac利用前提で記載しているため、Windowsユーザにはやりずらかったり、
当然実行はコマンドラインを使うのでインターネット利用のPCユーザには難しい。

ということで、

  • 「HPやブログを立ち上げたいがどうしたらよいですか？」

という質問をする人には、塩対応だったなと反省。
これからは立ち上げ最初は手伝おうと感じました。

っと、そんなとりとめのないことを書きましたが、
今後も少しずつ記載していこうかなと。
